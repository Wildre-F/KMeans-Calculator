##  About

An interactive K-Means Clustering calculator built as part of my Machine Learning module. It visually steps through each iteration of the algorithm — showing the distance tables as they update and rendering the final cluster graph, making it easy to understand exactly how K-Means works under the hood.

##  Features

| Feature | Description |
|---|---|
| 📊 **Iteration Tables** | View distance calculations updating at each iteration |
| 🎯 **Centroid Tracking** | Watch centroids move and recalculate in real time |
| 📈 **Final Cluster Graph** | Visual plot of all data points coloured by cluster |
| ⚙️ **Configurable** | Set your own number of clusters and data points |
| 🔄 **Convergence Detection** | Algorithm automatically stops when centroids stabilise |

##  How it works

1. Generate or input your data points
2. Set the number of clusters **(K)**
3. The algorithm randomly initialises centroids
4. Each iteration assigns points to the nearest centroid
5. Centroids recalculate based on cluster averages
6. Repeats until convergence — then renders the final graph
